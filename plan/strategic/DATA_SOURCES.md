# Reborn — 데이터소스 상세 스펙

> 각 외부 소스에서 가져올 수 있는 데이터 항목 정리 + Fiber 맵핑

---

## 1. 트렌드업 (TrendUp)

국내 소셜 데이터 리스닝 플랫폼. 네이버 블로그/카페, 인스타, 유튜브, 커뮤니티, 뉴스 등 국내 소셜 채널 커버리지 강점.

### 가져올 수 있는 데이터

| 데이터 항목 | 설명 | Fiber 맵핑 |
|------------|------|-----------|
| 버즈량 (볼륨) | 키워드별 일/주/월 언급 건수 추이 | 소셜 버즈 |
| 소셜 언급 원문 | 실제 게시물/댓글 원문 텍스트 + 출처 URL | 오디언스 보이스 |
| 감성 분석 | 긍정/부정/중립 비율 및 추이 | 감성 신호 |
| TPO 키워드 | 시간(Time)·장소(Place)·상황(Occasion) 연관어 | 맥락 키워드 |
| 채널별 분포 | 블로그/카페/인스타/유튜브/뉴스 등 채널 비중 | 소셜 버즈 (하위) |
| 인플루언서/주요 계정 | 해당 키워드 언급 상위 계정 | (Phase 2) |
| 연관 키워드 | 함께 언급되는 키워드 네트워크 | 맥락 키워드 |

### 수집 조건 (사용자 입력)
- **키워드** (필수): 브랜드명, 제품명, 캠페인명 등
- **기간** (필수): 시작일 ~ 종료일
- **채널 필터** (선택): 특정 채널만 수집

### 수집 방식
- 현재: 사용자가 트렌드업에 로그인 → 조건 설정 → 엑셀 다운로드 → Reborn 업로드
- 목표: 키워드+기간 입력 → 백엔드가 트렌드업에서 자동 크롤링
- ⚠️ 법적 검토 필요 (유료 이용 중이나 크롤링 허용 여부 서면 확인 필요)

## 2. 리스닝마인드 허블 (ListeningMind Hubble)

어센트코리아가 만든 검색 데이터 기반 소비자 의도(인텐트) 분석 플랫폼. 네이버/구글 검색어에서 소비자 의도와 관심사 흐름을 분석.

### 가져올 수 있는 데이터

| 데이터 항목 | 설명 | Fiber 맵핑 |
|------------|------|-----------|
| 검색량 추이 | 키워드별 네이버/구글 검색 볼륨 시계열 | 검색량 |
| 인텐트 분류 | 검색어의 의도 분류 (정보 탐색, 구매 의도, 비교 등) | 검색 의도 |
| 패스파인더 (검색 여정) | 사용자가 A 검색 후 → B → C로 이동하는 검색 경로 | 검색 여정 |
| 클러스터 분석 | 관련 검색어들의 의미 기반 군집화 | 연관 검색어 |
| 페르소나 뷰 | 검색 행동 기반 소비자 유형 분류 | 검색 의도 (하위) |
| 연관 키워드 맵 | 키워드 중심 연관어 네트워크 시각화 데이터 | 연관 검색어 |
| 시즌/트렌드 분석 | 검색량의 계절성·트렌드 패턴 | 검색량 (하위) |

### 수집 조건 (사용자 입력)
- **키워드** (필수): 브랜드명, 카테고리명, 제품명
- **검색 엔진** (선택): 네이버 / 구글 / 전체
- **기간** (선택): 최근 N개월

### 수집 방식
- 현재: 사용자가 허블에 로그인 → 분석 결과 다운로드 → Reborn 업로드
- 목표: 키워드 입력 → 백엔드가 허블에서 자동 크롤링
- ⚠️ 법적 검토 필요 (유료 이용 중이나 크롤링 허용 여부 서면 확인 필요)

## 3. 구글 트렌드 (Google Trends)

전 세계 구글 검색 관심도를 무료로 제공하는 서비스. 2025년 7월 공식 API(alpha) 출시.

### 가져올 수 있는 데이터

| 데이터 항목 | 설명 | Fiber 맵핑 |
|------------|------|-----------|
| 검색 관심도 (Interest over time) | 키워드별 상대적 검색 관심도 (0~100) 시계열 | 검색량 |
| 지역별 관심도 (Interest by region) | 국가/지역별 검색 관심 분포 | 검색량 (하위) |
| 관련 검색어 (Related queries) | Top / Rising 연관 검색어 목록 | 연관 검색어 |
| 관련 주제 (Related topics) | 연관 주제 카테고리 (Top / Rising) | 연관 검색어 |
| 실시간 트렌딩 검색어 | 최근 24시간 급상승 검색어 (국가별) | 🏠 홈 화면 트렌딩 위젯 |

### 수집 조건
- **키워드** (필수): 최대 5개 비교 가능
- **지역**: 글로벌 / 국가 / 지역
- **기간**: 최대 5년 (API 기준)
- **카테고리**: 선택 가능

### 수집 방식
- **Google Trends API (alpha, 2025.07 출시)**: 공식 API. 일/주/월/년 집계, 지역 필터, 5년 히스토리컬
- 대안: pytrends (비공식 Python 라이브러리) — 불안정하지만 무료
- 실시간 트렌딩: BigQuery 퍼블릭 데이터셋 또는 API로 접근

### 특이사항
- 검색 관심도는 **상대값** (0~100)이라 절대 볼륨이 아님 → AIM 연산 시 정규화 로직 필요
- 홈 화면 트렌딩 위젯에 "최근 24시간 급상승 검색어" 직접 활용 가능

## 4. 네이버 데이터랩 (Naver DataLab)

네이버 공식 검색 트렌드 서비스. 네이버 통합검색 기반 검색어 트렌드 + 쇼핑인사이트 제공. **공식 API 존재**.

### 가져올 수 있는 데이터

| 데이터 항목 | 설명 | Fiber 맵핑 |
|------------|------|-----------|
| 검색어 트렌드 | 키워드별 상대적 검색량 추이 (1~100) | 검색량 |
| 쇼핑인사이트 (카테고리) | 쇼핑 검색 트렌드, 카테고리별 인기 키워드 | 검색량 (하위) |
| 성별/연령대별 검색 비중 | 검색어의 인구통계적 분포 | 검색 의도 (하위) |
| 기기별 검색 비중 | PC / 모바일 비율 | 참고 데이터 |

### 수집 조건
- **키워드** (필수): 최대 5개 비교
- **기간**: 2016.01.01 ~ 현재
- **성별/연령대**: 필터 가능

### 수집 방식
- **네이버 통합 검색어 트렌드 API** (공식): 검색광고 API 계정으로 호출 가능
- **네이버 검색광고 API**: 연관 키워드 + 월간 검색량 (절대값) 제공
- Rate limit 존재하나 상업적 사용 가능

### 특이사항
- 구글 트렌드와 달리 **한국 시장 검색 데이터의 절대적 비중** (네이버 점유율)
- 검색광고 API 통해 **월간 검색량 절대값** 확보 가능 → AIM 연산에 직접 활용 가능
- 연관 키워드 API로 "함께 검색되는 키워드" 자동 추출 가능

## 5. X(트위터) 언급 via Grok API

xAI의 Grok API를 통해 X(트위터) 실시간 데이터에 접근. Grok은 X 데이터에 실시간 접근 권한을 가진 LLM이므로, 직접 크롤링 없이 LLM 응답 형태로 X 언급 데이터를 추출.

### 가져올 수 있는 데이터

| 데이터 항목 | 설명 | Fiber 맵핑 |
|------------|------|-----------|
| X 언급량 | 특정 키워드/브랜드의 X 멘션 규모 (Grok 요약) | X 언급 (via Grok) |
| X 언급 요약 | 주요 멘션 내용 요약, 감성 경향 | X 언급 (via Grok) |
| 주요 트윗/스레드 | 가장 인게이지먼트 높은 관련 트윗 정리 | X 언급 (via Grok) |
| 실시간 반응 트렌드 | 특정 이벤트/캠페인에 대한 실시간 X 반응 | X 언급 (via Grok) |
| 감성/톤 분석 | X 멘션의 전반적 감성 (Grok이 분석) | X 언급 (via Grok) |

### 수집 조건 (사용자 입력)
- **키워드/브랜드명** (필수)
- **기간** (선택): 실시간 ~ 최근 N일
- **언어** (선택): 한국어 / 영어 / 전체

### 수집 방식
- **Grok API** (xAI): LLM에게 프롬프트로 X 데이터 질의 → 구조화된 응답 수신
- 직접 크롤링이 아닌 **LLM 답변** 형태 (Grok이 X 데이터에 실시간 접근)
- API 비용: Grok 4.1 Fast 기준 $0.20/MTok(입력), $0.50/MTok(출력) — 비교적 저렴

### 특이사항
- Grok을 통해 받는 거라 **정확한 절대 수치보다는 요약/해석 중심** 데이터
- "X 언급 (via Grok)" 으로 Fiber명에 출처 표시
- 정량 정확도보다 **정성적 반응 요약에 강점** → 소셜 클러스터링 Loom의 보조 입력으로 적합
- 신규 가입 시 $25 무료 크레딧 제공, 이후 사용량 기반 과금
